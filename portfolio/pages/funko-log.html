Funko Log

Sistema web de e-commerce para loja de Funko Pop.

Sobre o projeto

Aplicação de e-commerce com catálogo de produtos colecionáveis, experiência de compra e gestão de pedidos.

Funcionalidades principais

  • Vitrine de produtos e categorias.
  • Carrinho de compras.
  • Gestão de pedidos.
  • Controle administrativo da loja.

Tecnologias

PHP, MySQL, Bootstrap, HTML e CSS.
