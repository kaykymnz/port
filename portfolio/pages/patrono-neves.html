Sobre o Projeto

O Patrono Neves utiliza o mesmo modelo de página para detalhar a plataforma de cursos online, destacando objetivos, funcionalidades e organização do sistema.

Objetivo

A ideia era desenvolver uma aplicação completa que permitisse o gerenciamento eficiente das negociações de veículos entre clientes, vendedores e administradores, ajudando na organização, no controle de estoque e na automatização dos processos.

Funcionalidades Principais

  • Visualização de veículos disponíveis pelo cliente
  • Formulário para manifestar interesse e iniciar uma negociação
  • Área exclusiva para os vendedores acompanharem e gerenciar as negociações
  • Dashboard para o administrador com controle de estoque de veículos
  • Atualização automática de status e cancelamento de negociações

Público-Alvo

O sistema foi desenvolvido para uma loja fictícia, simulando as rotinas e necessidades reais de uma revendedora de automóveis de pequeno a médio porte.
