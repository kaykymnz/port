Apolo Veículos

Sistema web para gerenciamento de negociações de automóveis.

Sobre o projeto

Projeto acadêmico de um sistema completo para gerenciar negociações de veículos entre clientes, vendedores e administradores.

Funcionalidades principais

  • Visualização de veículos disponíveis.
  • Abertura e acompanhamento de negociações.
  • Controle de estoque para administração.
  • Atualização de status das negociações.

Tecnologias

PHP, MySQL, JavaScript, HTML e CSS.
